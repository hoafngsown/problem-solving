# Fancy Form — Currency Swap

A React + TypeScript currency swap form built with Vite.

## Prerequisites

- Node.js >= 18
- npm or yarn

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

Open http://localhost:5173

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Type-check + production build |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview production build |

## Tech Stack

React 19, TypeScript, Vite, Tailwind CSS, React Hook Form + Zod, Framer Motion
